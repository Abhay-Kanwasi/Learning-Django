Loops

Normal for loop

{% for name in names %}
  • name : {{name}}
    • {% endfor %}

    For loop counter from 1

    {% for name in names %}

    {{forloop.counter}}: {{name}}

    {% endfor %}

    For loop counter from 0

    {% for name in names %}

    {{forloop.counter0}} : {{name}}

    {% endfor %}

    For loop revcounter(reverse counter to 1)

    {% for name in names %}

    {{forloop.revcounter}} : {{name}}

    {% endfor %}

    For loop revcounter(reverse counter to 0)

    {% for name in names %}

    {{forloop.revcounter0}} : {{name}}

    {% endfor %}

    For loop first (only first value will be true)

    {% for name in names %} {{forloop.first}} : {{name}} {% endfor %}

    For loop last (only last value will be true)

    {% for name in names %} {{forloop.last}} : {{name}} {% endfor %}
